HOW TO ACCESS SETTINGS
================================================================================

FROM APP:
1. Tap Profile tab (bottom right)
2. Tap gear icon (⚙️) in top-right corner
3. Select "⚙️ Cài đặt" from dropdown menu
4. Settings screen opens with full customization options

FROM CODE:
- Navigate to Settings via: navigation.navigate('Settings')
- Access useTheme() hook for theme colors
- Access useTranslation() hook for text translations

NEXT STEPS FOR FUTURE DEVELOPERS
================================================================================

TO ADD A NEW THEME:
1. Open src/config/themes.ts
2. Add new ThemeName type
3. Create new theme object with colors
4. Add to THEMES array

TO ADD A NEW LANGUAGE:
1. Create src/locales/[language].json
2. Translate all keys from en.json or vi.json
3. Update LocalizationContext to include new language
4. Add language option to SettingsScreen

TO CUSTOMIZE SETTINGS UI:
1. Edit src/screens/SettingsScreen.tsx
2. Modify styles.ts stylesheet
3. Add new sections as needed

TO DEBUG THEME ISSUES:
1. Check useTheme() hook in component
2. Verify THEMES[theme] has correct colors
3. Check AsyncStorage for saved preference
4. Review ThemeContext for state updates
